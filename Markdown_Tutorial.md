## **1. Heading 만들기**
문서에 헤딩을 추가하기 위한 구문은 다음과 같습니다.<br>
> \[ 글자 크기 레벨만큼의 # ] \[ 텍스트 \]

* 이 때, #들과 텍스트 사이에 ***공백*** 을 빼먹지 않도록 주의합시다!

예시로, 2만큼의 크기를 가진 헤딩을 만드는 코드는 다음과 같습니다.


> 예시 코드 :
> 
>      ## Hello World!
> 렌더링된 출력 : <h2>Hello World!

<br>
또한, h1 크기와 h2 크기의 경우 각각<br>
(===)와 (---)를 활용한 대체 코드를 사용할 수도 있습니다. <br><br>
코드의 형태는 다음과 같습니다.

> 예시 코드 :
>
>       Hello World!      
>       ============ 
> 렌더링된 출력 : <h1>Hello World!

<br><br><br><br>

## **2. 단락 만들기**
문서에 단락을 만드려면 빈 줄을 추가하면 됩니다.
* 단, 단락에 ***들여쓰기*** 를 하면 안 됩니다!
<br>

> 예시 코드 :
>
>       Hello World!
>
>       Good Bye!
> 렌더링된 출력 :
>
> Hello World!
>
> Good Bye!

<br><br><br><br>

## **3. 줄바꿈**
줄바꿈을 하고 싶다면 라인이 끝났을 때 최소 **2개의 빈 공간**을 스페이스바로 생성하고 엔터키를 입력하여 다음으로 작성할 라인으로 넘어가면 됩니다. <br>
이 방법의 경우 줄바꿈의 여부를 확인하기 힘들 수 있습니다. 이에 대해 Markdown에서는 줄바꿈을 **\<br>** 태그로 수행하는 것을 허용하고 있습니다.

> 예시 코드 :
>
>       This is the first line. (최소 2개의 빈 공간)
>       And this is the Second line.
>혹은,
>
>       This is the first line. <br>
>       And this is the Second line.
>렌더링된 출력 :
>
>This is the first line.
>
>And this is the Second line.

## **4. 텍스트에 변화 주기**
Markdown 문서 작성자는 강조하고 싶은 단어를 굵게 하거나 기울일 수 있습니다.
<br><br>

### **4-1. 텍스트 굵게 하기**
텍스트를 굵게 하고 싶다면 해당 텍스트를 * 혹은 (_) 두 개로 묶으면 됩니다.

* 단, (_)는 단어 중간에 있는 텍스트를 강조하지 못 합니다.

> 예시 코드 :
>
>       Oh **my** god     혹은    Oh __my__ god
>       Tyrann**osa**urus
>렌더링된 출력 :
>
>oh **my** god
>
>Tyrann**osa**urus


<br><br>

### **4-2. 텍스트 기울이기**
텍스트를 기울이고 싶다면 해당 텍스트를 * 혹은 (_) 하나로 묶으면 됩니다.

* 단, 마찬가지로 (_)는 단어 중간에 있는 텍스트를 강조하지 못 합니다.

> 예시 코드 :
>
>       Oh *my* god     혹은    Oh _my_ god
>       Tyrann*OSA*urus
>렌더링된 출력 :
>
>oh *my* god
>
>Tyrann*OSA*urus 

<br><br>

### **4-3. 텍스트 굵고 기울게 바꾸기**

텍스트를 굵고 기울게 만들고 싶다면 해당 텍스트를 * 혹은 (_) 세 개로 묶으면 됩니다.

* 이번에도 (_)는 단어 중간에 있는 텍스트를 강조하지 못 합니다.

> 예시 코드 :
>
>       Oh ***my*** god     혹은    Oh ___my___ god
>       Tyrann***OSA***urus
>렌더링된 출력 :
>
>oh ***my*** god
>
>Tyrann***OSA***urus

<br><br><br><br>

## **5. 인용**
인용구를 만들려면 단락 앞에 > 기호를 추가하면 됩니다.

> 예시 코드 :
>
>       >코로나 언제 끝나
>렌더링된 출력 :
>
>>코로나 언제 끝나

<br><br>

### **5-1. 여러 단락의 인용구**
여러 단락의 인용구를 만드려면 단락들의 사이에 > 만 있는 라인을 추가하면 됩니다.

> 예시 코드 :
>
>       >코로나 대체
>       >
>       >언제 끝나
>렌더링된 출력 :
>
>>코로나 대체
>>
>>언제 끝나

<br><br>

### **5-2. 중첩 인용구**
인용구 안에 인용구를 추가하고 싶다면 중첩하려는 단락 앞에 > 대신 >>를 추가하면 됩니다.

> 예시 코드 :
>
>       >군대는
>       >
>       >>언제 가지
>렌더링된 출력 :
>
>> 군대는
>>
>>>언제 가지

<br><br><br><br>

## **6. 리스트**
### **6-1. 숫자로 정렬된 리스트**
순서가 있는 리스트를 만들려면 다음과 같은 형식으로 만들 수 있습니다.
> 1\. \[ 텍스트 \]
>
> [숫자]. [ 텍스트 ]
>
> [숫자]. [ 텍스트 ]
>
> <ol>1. [ 텍스트 ]
>
> [숫자]. [ 텍스트 ] </ol>
> [숫자]. [ 텍스트 ]

* 정렬된 리스트의 맨 처음 순서의 맴버는 무조건 1로 시작해야 합니다. 이후의 숫자는 임의로 배정되므로 아무 수나 넣어도 상관 없습니다.
* 어떤 리스트의 하위 리스트를 만들고 싶다면 위의 형식처럼 들여쓰기(텝 또는 공백 4개)를 넣어주면 됩니다.

<br>

### **6-2. 정렬되지 않은 리스트**
정렬되지 않은 리스트는 다음과 같은 형식으로 만들 수 있습니다.
> ( - , * , + 중 한 가지를 골라서 ... )
>
> * [텍스트]
> * [텍스트]
>   * [텍스트]
>   * [텍스트]
> * [텍스트]


<br><br><br><br>

## **7. 리스트에 요소 추가하기**
작성자가 원한다면 리스트 안에 다른 요소를 추가할 수도 있습니다.

<br><br>

### **7-1. 리스트에 또 다른 리스트 추가하기**
챕터 6에서 이미 다루었듯이, 리스트 안에 또 다른 리스트를 추가할 수도 있습니다.<br>
추가하고 싶은 맴버 아래에 들여쓰기(탭 또는 공백 4개)를 넣고 다른 리스트를 만들면 됩니다.
<br>이 때, 정렬되지 않은 리스트의 하위에 정렬된 리스트를 추가할 수도 있고, 반대의 경우도 가능합니다.

<br><br>

### **7-2. 리스트에 단락 추가하기**
리스트에 단락을 추가하고 싶다면 두 라인을 공백으로 두고, 그 사이에 들여쓰기를 한 번 한 후 단락을 작성하면 됩니다.

> 예시 코드 :
>
>       1. 사과
>       2. 우유    
>       
>           1000mL짜리 두 개 사오기
>
>       3. 계란
>
>렌더링된 출력 :
>
>1. 사과
>2. 우유
>
>   100mL짜리 두 개 사오기
>
>3. 계란

<br><br>

### **7-3. 리스트에 인용구 추가하기**
리스트에 인용구를 추가하고 싶다면 두 라인을 공백으로 두고, 그 사이에 들여쓰기를 한 번 한 후 인용구를 >와 함께 작성하면 됩니다.

> 예시 코드 :
>
>       1. 사과
>       2. 우유    
>       
>           > 1000mL짜리 두 개 사오기
>
>       3. 계란
>
>렌더링된 출력 :
>
>1. 사과
>2. 우유
>
>   > 100mL짜리 두 개 사오기
>
>3. 계란


<br><br>

### **7-4. 리스트에 코드 블록 추가하기**
리스트에 코드 블록을 추가하려면, 두 라인을 공백으로 두고, 그 사이에 들여쓰기를 **두 번** 한 후 코드를 작성하면 됩니다.

> 예시 코드 :
>
>       1. Hello
>       2. World
>
>               printf("Hello World!!!");
>
>       3. !!!
>렌더링된 출력 :
>
>1. Hello
>2. World
>
>        printf("Hello World!!!");
>
>3. !!!

<br><br>

### **7-4. 리스트에 이미지 추가하기**
리스트에 이미지를 추가하려면, 두 라인을 공백으로 두고, 그 사이에 들여쓰기를 한 번 한 후 이미지의 형식대로 이미지를 넣어주면 됩니다.

> 예시 코드 :
>
>       이건 초코우유를 훔쳐 달아난 범인을 잡기 위한 단서들입니다.
>       1. 범인은 초코우유를 완전 좋아할거라 추측할 수 있습니다.
>       2. 범인이 만진 다른 초코우유에서 지문을 찾았습니다.
>
>           ![print](IMG/muyaho.jpg "지문")
>
>       3. 목격자의 증언에 의하면 범인은 초콜렛 맛이 난다고 합니다.
>렌더링된 출력 :
>
> 이건 초코우유를 훔쳐 달아난 범인을 잡기 위한 단서들입니다.
> 1. 범인은 초코우유를 완전 좋아할거라 추측할 수 있습니다. 
> 2. 범인이 만진 다른 초코우유에서 지문을 찾았습니다.
>
>    ![print](IMG/muyaho.jpg "지문")
>
> 3. 목격자의 증언에 의하면 범인은 초콜렛 맛이 난다고 합니다.

## **8. 코드**
단어나 구를 코드체로 표기하려면 (\`) 키로 묶으면 됩니다.
(\`) 키는 키보드 맨 왼쪽 위에 있습니다.

> 예시 코드 :
>
>       C에서 화면에 무언가를 출력하는 함수는 `printf()`입니다.
>
>렌더링된 출력 :
>
> C에서 화면에 무언가를 출력하는 함수는 `printf()`입니다.

* 이 때, 코드로 표시하려는 단어나 구에 이미 (\`)가 있을 경우에는 (\`) 대신 (\`\`)으로 묶으면 됩니다.

<br><br>
코드 블록을 만들고 싶다면 모든 줄에 들여쓰기를 넣어주면 됩니다.

> 예시 코드 :
>
>       C에서 Hello World! 를 화면에 출력하는 코드 :
>           #include <stdio.h>
>           int main(void)
>           {
>               printf("Hello World!);
>               return 0;
>           }
### 렌더링된 출력 :

C에서 Hello World! 를 화면에 출력하는 코드 :

    #include <stdio.h>
    int main(void)
    {
        printf("Hello World!);
        return 0;
    
    }

## **9. 수평선 넣기**
수평선은 (*) 이나 (-), (_) 중 하나를 골라 한 라인을 단독으로 3개 이상 쭈욱 늘이면 됩니다. 이 때, 호환성을 위해 수평선 코드 위 아래는 비워놓는 것이 좋습니다.

> 예시 코드 :
>
>       우와 수평선이다
>
>       ******************
>
>       우와아아
>
>렌더링된 출력 :
>
>우와 수평선이다
>
>***********
>
> 우와아아

## **10. 링크**
하이퍼링크 텍스트를 만들려면 다음 형식대로 코드를 작성하면 됩니다.

> \[ 이름 ]\(UPL "텍스트")

* \[이름]은 하이퍼링크를 저장할 텍스트입니다.
* "텍스트" 부분은 없어도 되나, 이를 작성할 경우 마우스를 가져다 대면 표시되는 텍스트를 만들 수 있습니다.

그냥 UPL만 나타내거나 이메일 주소를 링크로 빠르게 나타내려면 < >로 묶으면 됩니다.

ex ) \<https://...>  \<20201896@ssu.ac.kr>


<br><br>

### **10-1. 링크 서식 바꾸기**
링크도 강조하거나 기울일 수 있으며, 코드체로 변환할 수 있습니다.

> 예시 코드 :
>
>       **[example](https://example.org)**
>       *[example](https://example.org)*
>       [`example`](https://example.org)
>
>렌더링된 출력 :
>
> **[example](https://example.org)**
> *[example](https://example.org)*
> [`example`](https://example.org)

<br><br>

### **10-2. 참조 스타일 링크**
참조 스타일의 링크는 다음 두 코드의 세트 형식으로 구성됩니다.

> \[마우스를 대면 표시될 이름]\[라벨]
>
> ...
> 
> \[라벨]: \<UPL> "마우스를 대면 표시될 이름"

<br>
기존의 링크는 엄청난 길이의 UPL을 작성하던 단락에 우겨넣어야 한다는 단점이 있었으나, 참조 스타일 링크를 사용하면 보다 깔끔하게 단락을 작성할 수 있습니다.
<br>
<br>

> 예시 코드 :
>
>     단락 적을 게 마땅히 없으니 제 [GitHub repository UPL][1] 이라도 여기에 적어보겠습니다.
>
>     [1]:<https://github.com/hus0702/SE-Task-20201896> "GitHub repository UPL"
>
>렌더링된 출력 :
>
> 단락 적을 게 마땅히 없으니 제 [GitHub repository UPL][1] 이라도 여기에 적어보겠습니다.
>
> [1]:<https://github.com/hus0702/SE-Task-20201896> "GitHub repository UPL"

<br>
또한, 두 번째 코드의 경우 세 가지 규칙에 따라 전혀 다른 형태들이 나올 수 있습니다.

1. \[라벨]: 뒤에 공백 하나 이상 붙이기.
2. URL 꺽새< >로 묶어도 되고 안 묶어도 됨.
3. 마우스를 대면 표시될 이름은 ", ', () 셋 중 아무거나 골라서 묶어도 됨.

* UPL에 공백이 있는 경우, Markdown은 이를 인지하지 못 합니다. 따라서 공백 부분은 %20으로 대체해야 합니다.

## **11. 이미지**

<br>
이미지는 다음 형식으로 넣을 수 있습니다.

<br>

>
>       ![Image Example](IMG/f045e695a4b85a4c47c62dd402410b89.jpg "뿌직")
>
>렌더링된 출력 :
>
> ![Image Example](IMG/f045e695a4b85a4c47c62dd402410b89.jpg "뿌직")

## **12. 이스케이프 문자**

백 슬래시(\\)를 활용하면 여러 문자를 이스케이프할 수 있습니다.
* \\
* ` (백틱)
* \*
* _
* {}
* []
* <>
* ()
* \#
* \+
* \-
* .
* !
* |(파이프)
